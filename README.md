# Workout planner

## Technologies

- Python 3.11
- Streamlit
- OpenAI GPT-3

## Installation requirements

```bash
pip install -r requirements.txt
cp ./.streamlit/secrets.toml.example ./.streamlit/secrets.toml
# Update the API keys in ./.streamlit/secrets.toml
```

## How to run the code

```bash
streamlit run main.py
```
